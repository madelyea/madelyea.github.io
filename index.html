Portfolio

Below are a few of the projects I've completed.

Grateful Dead Web Application

In this project, I worked with Issac Greenfield to create a web application that dynamically loads taped shows from the current day in history using Archive.org's JSON API. Selecting a show takes you to an embedded playback feature.

Vyper Smart Contract

This was the final project I did for my Blockchain Development and Security course. I demo a trustfund smart contract I wrote in Vyper using a screencast. Much of this course was spent discovering vulnerabilities in Solidity, which is why our final project was written in Vyper, a safer language for smart contracts.

Artificial Intelligence Project

In this Python project, Robby the Robot learns to explore a ten by ten grid using Q-learning.

Operating Systems Project

In this project, I expanded on the existing xv6 operating system architecture (C Language). This was for my Introduction to Operating Systems course. I learned the core concepts of operation systems, including processes and threads, scheduling, synchronization, memory management, file systems, and the I/O subsystem.

Computer Vision Project

This is a project I completed in my Computer Vision course. It takes in two images with faces and morphs them by matching feature points and utilizing Delauney Triangulation. This is repeated until there are enough images to generate a blended gif.

Quicksorts

An Implementation of Quicksort with Insertion Sort, Dual Pivot Quicksort, and Dual Pivot Quicksort with Insertion Sort. This was a group project in my Algorithms course. We compared the search times of these three algorithms.

Gaussian Elimination

This is a simple project written in Python that I completed in my Numerical Computations course. It takes in a matrix and solution list and performs Gaussian Elimination.

Haskell Factorial

A program written in Haskell that solves the factorial of a number. This was a project in my Programming Languages course.

Newton's Method

This program written in Java solves Newton's Method for complex numbers.
